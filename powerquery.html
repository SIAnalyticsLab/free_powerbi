Power Query Data Cleaning Checklist

A clean, step-by-step guide for first-time users — Powered by SI Analytics

1. Remove Duplicates

Steps

1. Select the column(s).
2. Go to Home → Remove Rows → Remove Duplicates.
3. Power Query keeps the first occurrence and removes the rest.

2. Change Data Types

Steps

1. Select the column.
2. Go to Home → Data Type.
3. Select Text, Number, Date, Date/Time, etc.

3. Extract Year / Month from Dates

Steps

1. Select your date column.
2. Go to Add Column → Date.
3. Choose Year, Month, or Day.

4. Split Column by Delimiter

Steps

1. Select the column.
2. Go to Home → Split Column → By Delimiter.
3. Select comma, space, semicolon, or custom delimiter.

5. Fill Down / Replace Values

Fill Down

1. Select column.
2. Go to Transform → Fill → Down.

Replace Values

1. Select column.
2. Go to Transform → Replace Values.
3. Enter find + replace text.

6. Trim & Clean Text Fields

Steps

1. Select the column.
2. Go to Transform → Format → Trim.
3. Repeat with Clean.

7. Append Queries (Stack Tables)

Steps

1. Go to Home → Append Queries.
2. Select the tables you want to stack.
3. Click OK.

8. Merge Queries (Join Tables)

Steps

1. Go to Home → Merge Queries.
2. Choose the second table.
3. Select matching columns.
4. Choose the join type (Left join by default).
5. Expand the merged table to bring in new fields.
