‚≠ê What Calculation Groups Can You Create Using the GDP Dataset?

Calculation Groups allow you to apply time-intelligence logic to ANY measure you already built. Today, you will create a Time Intelligence Calculation Group for GDP trends.

Create a Calculation Group called: "GDP Time Intelligence"

Inside it, create the following calculation items:

1Ô∏è‚É£ Current Year GDP

SELECTEDMEASURE()

2Ô∏è‚É£ Previous Year GDP

CALCULATE(
    SELECTEDMEASURE(),
    SAMEPERIODLASTYEAR('gdp'[year])
)

3Ô∏è‚É£ GDP Year-over-Year Growth (%)

DIVIDE(
    SELECTEDMEASURE() - CALCULATE(SELECTEDMEASURE(), SAMEPERIODLASTYEAR('gdp'[year])),
    CALCULATE(SELECTEDMEASURE(), SAMEPERIODLASTYEAR('gdp'[year]))
)

4Ô∏è‚É£ GDP Year-over-Year Difference

SELECTEDMEASURE()
    - CALCULATE(SELECTEDMEASURE(), SAMEPERIODLASTYEAR('gdp'[year]))

5Ô∏è‚É£ Moving Average (3-Year)

CALCULATE(
    SELECTEDMEASURE(),
    DATESINPERIOD('gdp'[year], MAX('gdp'[year]), -3, YEAR)
)

You can now apply these calculations to any base measure such as:

  • Total GDP
  • Average GDP
  • Max GDP
  • Min GDP
