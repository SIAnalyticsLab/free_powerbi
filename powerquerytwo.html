1. Remove Duplicates

How to Do It

1. Select your column(s).
2. Go to Home ‚Üí Remove Rows ‚Üí Remove Duplicates.
3. Power Query keeps the first unique row.

2. Change Data Types

How to Do It

1. Select a column.
2. Go to Home ‚Üí Data Type.
3. Pick Text, Number, Date, etc.

3. Extract Year/Month

How to Do It

1. Select your Date column.
2. Go to Add Column ‚Üí Date.
3. Choose Year, Month, Day.

4. Split Column by Delimiter

How to Do It

1. Select the column.
2. Go to Home ‚Üí Split Column ‚Üí By Delimiter.
3. Choose comma, space, semicolon, custom.
